# `ppk_to_openssh.py`

A Python script to convert **PuTTY PPK (PuTTY-User-Key-File-3)** format private keys to **OpenSSH** private key format.

## Overview

This utility parses PuTTY's `.ppk` key files, decodes the embedded key material, and exports them in a format compatible with OpenSSH tools (e.g., `ssh`, `scp`, `git`). It supports keys generated by recent versions of PuTTY (PPK version 3).

> ⚠️ **Note**: This script currently supports **unencrypted** PPK files. Support for passphrase-protected keys may be added in future versions.

## Requirements

- Python 3.6+
- `cryptography` library

Install dependencies:

```bash
pip install cryptography
```

## Usage

```bash
python3 ppk_to_openssh.py <input_ppk_file> <output_openssh_file>
```

### Example

```bash
python3 ppk_to_openssh.py mykey.ppk id_rsa
```

This will generate an OpenSSH-compatible private key file named `id_rsa`.

## Permissions

After conversion, secure the private key file:

```bash
chmod 600 id_rsa
```

## How It Works

1. **Parses** the PPK file header and extracts metadata (key type, encryption status, comment).
2. **Decodes** base64-encoded public and private key blobs.
3. **Reconstructs** the key using the `cryptography` library.
4. **Exports** the key in OpenSSH private key format.

## Limitations

- Only supports **PPK version 3**.
- Does **not yet support** passphrase-protected PPK files.
- Currently tested with **RSA** keys. ED25519 and other key types may require additional handling.
